# 📖 Overview

The **Personal Finance Tracker** is a command-line application designed to help users manage their daily income and expenses.

The project demonstrates:

* Object-Oriented Programming (OOP)
* CSV File Handling
* Input Validation
* Data Persistence
* Clean & Modular Code

---

# ✨ Features

✅ Add Credit Transactions

✅ Add Debit Transactions

✅ Automatically Record Current Date

✅ Store Transactions in CSV File

✅ View Complete Transaction History

✅ Generate Financial Statistics

✅ Input Validation

✅ Exception Handling
